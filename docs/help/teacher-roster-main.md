## What this is

The Teacher Roster is your **one screen for every student in your
class**, with all the things you usually need at a glance: name, grade,
program flags (ESE / 504 / ELL), FAST scores, hall pass / tardy
counters, and safety-plan indicators.

## Top of the page

- **Section picker**: switch between class periods if you teach more
  than one. Core Team members see a "View any teacher's roster"
  toggle.
- **View tabs**: Main • Benchmark • Spider • Insights •
  Accommodations. (Each has its own help article — ask me about any
  of them.)
- **Search box**: filter the visible roster by name.

## Reading a student row

- **Name + grade**: click the name to open the student's full profile.
- **Flags column**: small badges for ESE, 504, ELL, Safety Plan,
  MTSS Tier 2/3, etc. Hover for the detail.
- **FAST column**: latest PM-window score + level (color-coded
  L1–L5). A green check ✓ means the student showed Learning Gains on
  the most recent PM.
- **HP / Tardy counters**: rolling 30-day totals. Click the number to
  see the detail log.

## Quick actions per row

- **+ Pass**: send a hall pass for that student without leaving the
  page.
- **+ PBIS**: award points / give a recognition.
- **Note**: add a quick staff note (private to staff).
- **Edit**: opens the student profile in a side drawer for fast edits.

## Sorting & exports

- Click any column header to sort.
- The **⋯** menu top-right has **Export CSV** and **Print roster**.

## Tip

The roster is the launchpad for almost every other feature — most
deep dives (Benchmark, Spider, Insights, Accommodations) start by
picking the section here, then switching tabs.
