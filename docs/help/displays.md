## What this is

**Displays** is the hub for every TV / signage screen in the building
— hallway monitors, cafeteria TVs, classroom info screens. You build
**playlists** (image / video / audio / PDF) and **schedule** them to
specific screens.

## The three pieces

1. **Items** — your media library. Upload images, MP4s, MP3s, PDFs.
2. **Playlists** — ordered lists of items with per-item duration.
3. **Screens** — physical displays in the building, each pointed at
   one playlist (or a schedule of playlists by time of day).

## Building a playlist

1. **Displays → Playlists → + New Playlist**.
2. Name it, pick the school (admins managing multiple) or auto if
   single-school.
3. **+ Add item** — pick from library or upload new.
4. Drag to reorder. Set per-item display time (image default 8s,
   video plays through).
5. **Save**.

## Pointing a screen at a playlist

1. **Displays → Screens** → pick a screen.
2. Set **Active playlist** (always-on) OR build a **Schedule**
   (e.g. "Morning playlist 7–9am, Lunch playlist 11–1pm").
3. Save — the screen picks it up on next refresh (within ~30s).

## Special tiles you can mix in

- **House standings** — live PBIS house scoreboard (auto-pulls).
- **Active hall passes** — current students out of class (great for
  front-office screens).
- **Heartbeat tile** — school-wide PBIS / attendance pulse.
- **Pickup queue tile** (if Pickup module is on) — classroom screens
  show "Your car is here" filtered to that roster.

## Setting up a new TV

1. Plug in a Chromecast / Firestick / repurposed laptop.
2. Open the kiosk URL shown under **Screens → + Add screen** on the
   TV's browser.
3. Enter the pairing code. The TV is now a registered screen and will
   play whatever you assign it.

## Common situations

- **Need to push an emergency announcement**: build a single-item
  playlist with the message image, switch all screens to it, then
  swap back when done.
- **PDF not displaying**: PDFs render page 1 only. Split multi-page
  docs into separate items or convert to images.
- **Audio plays from one TV in the hallway**: each screen has a
  **Mute** toggle — turn off audio on all but one.

## Permissions

- **Teachers**: can build playlists in their own classroom scope.
- **Admin / front-office**: full school-wide control.
