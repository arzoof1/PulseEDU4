## What this is

The **Class Detail** accommodations page is your **legal record** that
required accommodations are being implemented. For each student in your
class who has an IEP, 504, ELL, or other plan, you see the exact
accommodations and can log when you've provided them.

This is the screen auditors / parent advocates / due-process hearings
will eventually ask about — keep it current.

## Layout

- **Rows = students** with accommodations (others hidden).
- **Columns = accommodations**, grouped by category (Presentation,
  Response, Setting, Timing/Scheduling).
- Each cell shows the student's logged frequency: ✓ today, ✓ this
  week, last-N-days bar, etc.

## Logging an accommodation

1. Click the cell for the student + accommodation you provided.
2. Pick **today** (default) or back-date.
3. Optionally add a short note (e.g. "extended time on Ch 4 quiz").
4. Save.

Bulk option: **Log all for today** at the top of any column logs that
accommodation for every student listed under it.

## Filters

- **Show only**: ESE / 504 / ELL / All
- **Subject** (if you teach multiple)
- **Last N days**: how far back to count "implemented"

## Reading the color bar

- **Green** — logged today
- **Light green** — logged this week
- **Yellow** — logged in the last 2 weeks
- **Red** — over 2 weeks since last log → action needed

## Common uses

- **Friday wrap-up**: 2-minute pass to bulk-log routine accommodations
  for the week.
- **Pre-IEP meeting**: print the student's column to show frequency
  of implementation.
- **Sub coverage**: a sub can see today's required accommodations at
  a glance.

## Privacy

Accommodation detail is staff-only. The parent portal shows only that
the student has accommodations, not the specifics, unless your admin
has opted in.
