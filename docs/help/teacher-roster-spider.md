## What this is

The **Spider** (radar chart) report gives you a visual "shape" of each
student's strengths and weaknesses across FAST **reporting categories**
— the higher-level groupings above individual benchmarks.

A balanced student has a roughly round shape; a student with a glaring
weakness has a deep dent in one or two spokes.

## How to read one chart

- **Each spoke = one reporting category** (e.g. for 5th Math:
  Number Sense & Operations, Algebraic Reasoning, Measurement & Data,
  Geometric Reasoning).
- **Distance from center = % mastery** on that category for the
  selected PM window.
- **Colored band** behind the shape is the class average — quick way
  to spot a student well below their peers on one spoke.

## Layout

- **All students at once**: small-multiples grid — every kid in the
  section gets a thumbnail spider. Great for skim-comparing the whole
  class at PM3.
- **Single student**: click any thumbnail to enlarge, with the exact
  % per spoke listed beside the chart.

## Filters

- **PM window**: PM1 / PM2 / PM3. Hold ⌘/Ctrl to overlay PM1 vs PM3
  on the same chart to see growth.
- **Subject**: ELA / Math / Science.
- **Show class avg**: toggle the comparison band on/off.

## Common uses

- **Parent conferences**: print one student's spider with the class
  average band — visual + plain-language.
- **MTSS referrals**: a student whose shape is small and lopsided
  is a stronger Tier 2 candidate than one who's small but evenly low.
- **Track growth**: PM1 vs PM3 overlay shows whether your reteach
  cycle actually moved the needle.

## Print / export

⋯ menu → **Print single** (current selected student) or
**Print all** (one student per page, full section).
